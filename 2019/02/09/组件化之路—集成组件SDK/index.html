组件化之路—集成组件SDK

发表于 2019-02-09 |

介绍

组件化的前提是要有基础组件、功能组件、业务组件这三大块。其中基础组件和功能组件都可以做成SDK，可以供其他APP选择性的调用。

比如把地图组件单独封装成一个SDK，需要使用地图就加载这个SDK，不需要使用的就不加载。对于全部封装成一个公共库的做法，这样既能实现解耦，又可以减少包的大小。
